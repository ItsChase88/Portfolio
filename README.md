
# Dynamic Portfolio

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/)

A dynamic portfolio system used to showcase work I have completed throughout my entire career and all of my professional achievements.

## The Motivation

While reviewing candidate resumes, I noticed a recurring trend: the most impactful candidates used polished portfolios.

I decided it was time to build my own, but with a twist. I wanted to avoid the "static site trap" where updating a single project requires a full code commit and deployment. This project was born from the need for a flexible, scalable, and dynamic platform to showcase my experience, skills, and education.
## Tech Stack

**Front End:** MudBlazor

**Back End:** Blazor (.NET 9)

**Database:** SQLite (Entity Framework)
